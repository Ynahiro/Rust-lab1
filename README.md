# Rust-lab1

Прянчиков Александр Евгеньевич

В работе изучается использование структур - пользовательских типов данных, позволяющих называть и упаковывать вместе несколько связанных значений.
Помимо создания и взаимодействия со структурами, также изучаестя возможность создания методов, для указания поведения, ассоциированного с типом структуры.
